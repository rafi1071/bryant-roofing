# Bryant Roofing Website

A premium roofing website with a brand-focused homepage and a high-converting landing page optimized for ads traffic, serving Houston County, Texas.

## Run Locally

**Prerequisites:** Node.js (v18+)

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

4. **Preview the production build locally:**
   ```bash
   npm run preview
   ```
